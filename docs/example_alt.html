Equations

Lesson 1

Copied from the Haese Year 7 textbook

Learning Outcome

distinguish between algebraic expressions where pronumerals are used as variables, and equations where pronumerals are used as unknowns

An equation is a mathematical sentenace which indicates that two expressions have the same value. An equation always contains an equals sign \((=)\).

A simple equation may be a true numerical statement like: \[\underbrace{4\times7}_{\textbf{LHS}}\enspace=\enspace\underbrace{17+11}_{\textbf{RHS}}\] Notice that an equation has a left hand side (LHS) and a right hand side (RHS) and these are separated by the equals sign.

An algebraic equation has an unknown or variable in it.

For example, \(\enspace4x+12=32\enspace\) contains the variable \(x\).

When we solve an equation we find the value of the variable which makes the equation true. This value is called the solution of the equation.

If we were to replace \(x\) by a variety of numbers, most of them would make the equation false.

For example, consider the equation \(\enspace4x+12=32\):

If \(\enspace x=1\) the LHS is \(4\times1+12=\) \(1+12=13\qquad\)
If \(\enspace x=10\enspace\) the LHS is \(4\times10+12=\) \(40+12=52\qquad\)
If \(\enspace x=5\) the LHS is \(4\times5+12=\) \(20+12=32\qquad\)
So, if \(\enspace x=5\) then the LHS = RHS, and the equation is true.
We say \(\enspace x=5\) is the solution of the equation \(\enspace4x+12=32\).

Examples

Example 1

Question

What number can replace \(\square\) to make the equation true?

  1. \(\quad3+\square=10\)
  2. \(\quad3\times\square=18\)
  3. \(\quad20\div\square=4\)
  4. \(\quad\square-8=4\)

Show Solution

What number can replace \(\square\) to make the equation true?

  1. \(\begin{aligned}[t]3+\square&=10\\3+\color{rgb(221, 72, 30)}{7}&=10\\\text{so}\quad\square&=7\end{aligned}\)

  2. \(\begin{aligned}[t]3\times\square&=18\\3\times\color{rgb(221, 72, 30)}{6}&=18\\\text{so}\quad\square&=6\end{aligned}\)

  3. \(\begin{aligned}[t]20\div\square&=4\\20\div\color{rgb(221, 72, 30)}{5}&=4\\\text{so}\quad\square&=5\end{aligned}\)

  4. \(\begin{aligned}[t]\square-8&=4\\\color{rgb(221, 72, 30)}{12}-8&=4\\\text{so}\quad\square&=12\end{aligned}\)
